- 사진을 찍을 때 태양이나 밝은 광원을 직접 바라볼 때 생기는 현상
	- 게임이나 3D 이미지의 경우, 장면을 더 현실적으로 보이게 하는 효과
	- 렌즈 플레어의 모양을 결정하는 텍스터 이미지 필요
	- LensFlare 객체 생성 후, 광원의 위치에 놓고 장면에 추가하면 렌즈 플레어 효과를 줄 수 있음
	
- 속성
	- texture
		: 플레어의 모양을 결정하는 이미지
	- size
		: 플레어의 크기 설정
	- distance
		: 광원(0)~카메라(1) 사이 거리
		  => 렌즈플레어의 정확한 위치 지정 가능
	- blending
		: 복수의 텍스처를 설정하는 경우, 텍스처들을 합치는 방법 설정
		- lensFlare.add 멤버 함수를 통해 텍스처의 블렌딩 모드 지정
	- color
		: 플레어의 색상 결정
