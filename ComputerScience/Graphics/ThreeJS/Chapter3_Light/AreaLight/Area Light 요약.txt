- 직사각형의 영역으로 빛을 방출하는 광원
	- 기본 Three.js 라이브러리에서는 제공하지 않음
	- WebGLRenderer 사용 불가능
	  (why?) Area Light은 매우 복잡한 광원이라서 렌더러 객체에 매우 심각한 성능 저하 야기
	- WebGLDeferredRenderer 사용
		: 렌더링시, 장면을 여러 단계로 분리하는 접근 방식을 사용
		=> 더 복잡한 광원에 대해서 렌더링 가능
	- 방출된 빛은 객체에 닿을 때만 볼 수 있음
	  i.e 장면에서 광원 자체를 볼 수 없음
	  => 광원의 위치에 Plane/Box 메시를 추가하여 발광영역 시뮬레이션 가능
