# 코드 정보
- 각 주제에 대한 스크립트 파일
- 필요에 따라 스크립트 파일을 참조 가능하도록 환경 변수에 추가
  - .profile에 환경 변수 추가
    - MYBASHSCRIPTPATH=$HOME/my-bash-script
    - PATH=$PATH:$MYBASHSCRIPTPATH
  - 홈 디렉터리($HOME)에 my-bash-script 디렉터리 생성
    - 해당 디렉터리에 스크립트 파일 추가
